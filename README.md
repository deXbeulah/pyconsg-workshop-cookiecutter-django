## Pre-requisites

* git
* Github account
* Text editor or IDE (e.g. VSCode or PyCharm)

## Workshop

* Install Docker

* Install cookiecutter

* Bootstrap our project

```bash
cookiecutter https://github.com/pydanny/cookiecutter
```

* Github OAuth login
* Deploy to heroku
* (Optional) Make it HTTPS
